# Day 2 - Tool Using Agent

## Objective
Extend a rule-based agent to use external tools.

## Tools Implemented
- Calculator Tool
- Weather Tool-Uses OpenWeatherMap API to fetch real-time weather data.
- Text Summarizer
- Time Tool

## Architecture
Input → Tool Selection → Tool Execution → Output

## Key Concepts
- Tool abstraction
- Function calling
- Modular design

## How to Run
cd day2
python agent.py
