# Day 1 - Rule-Based AI Agent

## Objective
Build a simple rule-based AI agent using Python.

## Approach
The agent follows a modular architecture:

1. Input Handler – Takes user input
2. Decision Module – Identifies intent using keyword matching
3. Action Module – Executes the corresponding action

## Features
- Greeting response
- Basic arithmetic calculation
- Current date retrieval

## Tech Used
- Python
- Rule-based logic (no ML)

## How to Run
1. Navigate to day1 folder
2. Run: python agent.py
